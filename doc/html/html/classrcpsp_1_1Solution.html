Check if the solution is feasible solution for RCPSP. Tests are following:

  • compute objective value and compute resources consumption
  • check if each resource is respected

int [][] rcpsp.Solution.m_consumptionResource
private

Sum of resources used at each step time. For a solution, computational of each resource is done by method validate().

For performance reasons, it is better to update this element iteratively with method setConsumptionResource(int i, int date, int val).

This iteratively update is done in addActivity(int j, int time).
