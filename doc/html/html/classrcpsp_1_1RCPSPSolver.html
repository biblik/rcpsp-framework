Detailed Description

This class is the place where you should enter your code and from which you can create your own objects.

The method you must implement is solve(long) This method is called by the programmer after loading the data.

The RCPSPSolver object is created by the Main class. The other objects that are created in Main can be accessed through the following RCPSPSolver attributes:

  • m_instance : the Instance object which contains the problem data
  • m_solution : the Solution object to modify. This object will store the result of the program.

m_timeLimit : the maximum time limit (in seconds) given to the program.
