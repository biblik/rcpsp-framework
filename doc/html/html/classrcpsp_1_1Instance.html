Detailed Description

The Instance class allows to create an object that contains the data stored in a rcpsp file.

The class is created through its constructor that takes the data file as parameter. The data file is read and the data are stored in the Instance object. The data can then be access calling the object methods.

Warning
Do not modify this class.
Author
Fabien Lehuede / Damien Prot / Axel Grimault 2017
